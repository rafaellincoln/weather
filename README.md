# Aplicação para visualização do clima na localização atual

## Ferramentas utilizadas
- React (create-react-app)
- Axios
- openweathermap API

## Requisitos
- Node >= 16
- NPM >= 8

## Instalação
- Utilizando o `yarn`

```
yarn
```

- Utilizando o `npm`

```
npm install
```

## Utilização
```
yarn start
```
Ou
```
npm start
```

## Build
```
yarn build
```
Ou
```
npm run build
```
